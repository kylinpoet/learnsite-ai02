# 教师侧按班查看小组与共享文件

更新时间：2026-04-01  
状态：已实现并已通过接口测试、类型检查与构建验证

## 1. 功能目标

教师在工作台顶部选择班级后，需要马上看到该班的：

1. 小组数量与共享文件总数。
2. 每个小组的成员、组长、座位、签到状态。
3. 每个小组共享网盘的容量使用情况。
4. 小组共享文件列表，并支持教师直接下载。

这个能力是学生侧“小组页 + 小组共享网盘”的教师观察与管理入口，重点是“按班聚焦”和“课堂内可快速查看”。

## 2. 用户入口

前端入口：

1. 教师登录 `/login/staff`
2. 进入 `/staff/dashboard`
3. 在页面顶部选择班级
4. 页面中部的“按班查看小组与共享文件”区域会自动切换到当前班级

当前没有单独新建教师“小组管理页”，而是直接挂在教师工作台里，避免教师在上课时频繁跳页。

## 3. 后端接口

### 3.1 班级小组总览

接口：

`GET /api/v1/staff/classes/{class_id}/groups`

返回内容：

1. 班级基础信息
2. 小组数量
3. 小组共享文件总数
4. 每个小组的成员列表
5. 每个成员的座位与今日签到状态
6. 每个小组共享网盘的容量信息和文件列表

权限规则：

1. 教师只能查看自己负责班级的数据
2. 管理员可以查看任意班级
3. 非授权教师访问返回 `403`

### 3.2 教师下载小组共享文件

接口：

`GET /api/v1/staff/drives/files/{file_id}`

行为：

1. 仅允许下载“小组共享网盘”中的文件
2. 会校验该文件所属小组对应的班级权限
3. 已授权教师和管理员可下载
4. 非授权教师访问返回 `403`

## 4. 前端页面行为

文件：

`apps/web/src/modules/staff-dashboard/StaffDashboardPage.vue`

本次接入后的页面行为如下：

1. 顶部“班级与课程”区域继续作为课堂聚焦入口。
2. 切换班级后，座位图和“小组协作”区域会同步切换。
3. 小组卡片内展示：
   - 组名、组号、组长
   - 成员人数、已签到人数、未签到人数
   - 每个成员的学号、座位、机房、签到时间
   - 共享网盘容量使用情况
   - 最近共享文件，并支持教师下载
4. 点击“刷新小组”可重新拉取当前班级数据。

## 5. 关联数据模型

本功能涉及以下核心表：

1. `student_groups`
2. `student_group_members`
3. `drive_spaces`
4. `drive_files`
5. `class_seat_assignments`
6. `attendance_records`
7. `teacher_class_assignments`

数据关系说明：

1. 一个班级可有多个小组。
2. 一个学生同一时间只属于一个小组。
3. 一个小组对应一个共享网盘空间。
4. 教师查看小组时，成员座位来自班级座位绑定，签到状态来自当天签到记录。
