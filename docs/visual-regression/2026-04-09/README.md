## 验收清单

### 1) 登录页（学生 / 教师）

- [ ] 标题与说明文字层级清晰，主标题不挤压。
- [ ] 表单项纵向间距均匀，输入框高度与按钮高度一致。
- [ ] 主按钮悬停时有轻微上浮反馈，无突兀阴影跳变。
- [ ] 说明卡片与表单卡片圆角、边框、阴影风格一致。
- [ ] 信息提示（`el-alert`）边框完整，无裁切和色块突变。

### 2) 学生学习首页

- [ ] Hero 区块背景更柔和，按钮与文案视觉重心平衡。
- [ ] 概览指标卡密度适中，数值优先级高于说明文案。
- [ ] 待完成 / 已完成学案卡片信息分层清楚，卡内留白一致。
- [ ] 学生档案与得分模块边框、底色、阴影一致，不显“塑料感”。
- [ ] 移动宽度下（窄屏）卡片栈布局无挤压和错位。

### 3) 教师仪表盘

- [ ] Hero 背景高亮不过曝，操作按钮可读性稳定。
- [ ] 概览指标卡 icon/数值/说明层级清晰，卡片悬停反馈轻量。
- [ ] 课堂座位卡、分组卡、活动卡圆角与边框统一。
- [ ] 重点提示盒（hint-box）不喧宾夺主，但能被快速识别。
- [ ] 整页信息密度可连续浏览，不出现“拥挤块状”区域。
